# Phishing Awareness Simulation Using Social Engineering Techniques

## Overview

This project demonstrates an ethical phishing awareness simulation created for educational purposes. The objective is to help users recognize phishing attempts and improve cybersecurity awareness through a controlled and harmless simulation.

The project includes a sample phishing email, an HTML-based demonstration login page, awareness messages, and an analysis of user responses. No credentials are collected or stored during the demonstration.

---

## Objectives

- Understand phishing techniques and social engineering.
- Design a harmless phishing awareness email.
- Develop an HTML login page for demonstration.
- Measure user awareness through a simulated exercise.
- Analyze results and recommend preventive measures.

---

## Methodology

The project follows these steps:

1. Research phishing techniques.
2. Create a phishing awareness email.
3. Develop an HTML demonstration login page.
4. Conduct an awareness simulation with informed participants.
5. Record observations.
6. Analyze the results.
7. Recommend cybersecurity best practices.
